Download groundtruth annotation and put them in this folder.

The related gt image name list of the .t7 file in the 'data' folder is in the .txt file.

Each .mat file contains the 2D coordinate of room corners of the image.

For Stanford 2d-3d dataset, we use area 1, 2, 4, 6 for training, area 3 for validation, and area 5 for testing.
